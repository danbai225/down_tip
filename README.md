<p align="center">
这是一个任务栏工具，它基于 <b>Golang</b> 开发。使用了<a target="_blank" href="https://github.com/danbai225/tipbar">tipbar</a>开发。集成了一些小模块</p>

# 功能模块
- down 下班倒计时
- ip 展示当前ip
- keyLog 按键记录
- socket 集成 tcp、udp、http、websocket 客户端/服务端并支持代理的连接测试 目前只有tcp客户端
- weather 天气和预警
- socks5proxy 代理
- code2img 代码转图片
